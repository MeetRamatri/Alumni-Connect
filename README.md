# Alumni Connect

Alumni Connect bridges the gap between students and alumni, fostering a thriving community of mentorship and growth.

**Key Features:**

*   **Facilitate Mentorship:** Connects students with alumni using **MongoDB** for efficient data retrieval, enabling personalized guidance and professional networking through a robust directory.
*   **Streamline Recruitment:** Empowers alumni to post exclusive job openings and internships via a **React-based** portal, granting students direct access to career advancements.
*   **Showcase Campus Culture:** Displays active clubs and events through a dynamic **framer-motion** interface, keeping the institution's spirit alive digitally.
*   **Enable Real-time Collaboration:** Integrates **Socket.IO** for instant chat and **Calendly** for seamless meeting scheduling, ensuring meaningful interactions between members.
